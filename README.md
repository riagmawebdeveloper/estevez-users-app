# EstevezUsersApp


Este proyecto ha sido generado con  [Angular CLI](https://github.com/angular/angular-cli) version 19.2.10.

## Objetivo del proyecto
Desarrollar una aplicación básica en Angular que cumpla con los siguientes requisitos mínimos:

Pantalla de Login
Permitir inicio de sesión con usuario y contraseña (puede ser validación simulada o mock).
No es necesario implementar backend real.
Módulo de Clientes
Pantalla para dar de alta un cliente con los datos que tú decidas (por ejemplo: nombre, correo, teléfono, empresa, etc.).
Mostrar en una tabla o lista los clientes registrados.
Permitir eliminar o editar un cliente (opcional, suma puntos).
Requisitos técnicos básicos:
Proyecto en Angular
Estructura limpia y ordenada del código.
Uso de componentes y servicios.
Interfaz sencilla pero funcional (puedes usar Angular Material o Bootstrap si prefieres).

## Instalación y ejecución 

1. Clonar repositorio

```bash
git clone https://github.com/riagmawebdeveloper/estevez-users-app
cd estevez-users-app
```

2. Instalar dependencias

```bash
npm install
```


3. Ejecutar modo desarrollo

```bash
ng serve
ó
npm start
```

La aplicación deberá de correr en:

http://localhost:4200
